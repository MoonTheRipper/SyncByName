What It Does

  • Scans multiple source roots recursively.
  • Checks comparison roots by filename only.
  • Filters by extensions like mp4, mov, and mxf.
  • Previews missing files before any copy begins.
  • Copies into an output folder without overwriting existing files.

Why It Exists

Most sync tools care about path structure, timestamps, or mirroring. Sync by Name is for the narrower job: find filenames that do not exist anywhere in the comparison set and collect them safely.

Behavior

  • One-way workflow only.
  • No deletes, moves, or destructive sync.
  • No overwrite by default.
  • Optional source-root preservation to avoid collisions.
